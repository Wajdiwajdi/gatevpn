How to Use Your WireGuard Config
Download WireGuard - Install WireGuard on your device from wireguard.com
Generate Config - Click the button below to generate your configuration file
Copy Config - Copy the entire config text (starting with [Interface])
Import to WireGuard - Open WireGuard, create new tunnel, paste the config
Update Keys - Replace YOUR_PRIVATE_KEY_HERE and SERVER_PUBLIC_KEY_HERE with your actual keys
Connect - Activate the tunnel and you're connected to your VPN!
